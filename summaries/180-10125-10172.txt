This document is a collection of reports, memos, and fingerprint cards related to a security violation case file from the HSCA (House Select Committee on Assassinations) concerning the tampering of Kennedy autopsy photographs. It includes requests for FBI fingerprint analysis and security background checks on HSCA staff, a fingerprint comparison report from the FBI identifying prints found on a photo book and safe, and a description of an investigation into the incident, focusing on the involvement of a CIA employee named Regis Blahut. The document also includes a letter from Director of Security to Chief Counsel, Robert Blakey, discussing fingerprint cards, key prints, and time.
