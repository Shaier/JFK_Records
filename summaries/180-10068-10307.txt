This document is a Final Determination Notification from the Assassination Records Review Board (ARRB) regarding a set of Staff Payroll Records from the HSCA (House Select Committee on Assassinations). The document is marked "Postponed in Part" because it contains social security numbers, which have been redacted due to privacy concerns. The next review date is set for 2017. It includes payroll authorization forms, a memorandum about payroll certification, and a recommendation to hire Lisa Berlow.
